## Checklist

### Flows
- [ ] Onboarding rodziny — tworzenie, zaproszenia, limity planu
- [ ] Listy zakupów — CRUD, szybkie dodawanie, odhaczanie, kategorie
- [ ] Sumowanie składników — prezentacja zsumowanych jednostek
- [ ] Auth (login, WebAuthn, 2FA), settings, admin (jeśli applicable)
- [ ] Przepisy (gdy Faza 4+) — CRUD, dodaj do listy

### UX quality
- [ ] Responsive layout (mobile-first, breakpoints)
- [ ] Accessibility: labels, icon-only buttons, focus, keyboard
- [ ] Copy and i18n consistency (en/pl)
- [ ] Loading, empty, and error states
- [ ] Toasts and dialogs (confirm destructive actions)
- [ ] Tables: column visibility, sort, filters

### API ↔ UI
- [ ] Error messages surfaced clearly (validation, 4xx/5xx)
- [ ] Pagination / long lists
- [ ] Optimistic updates vs stale data after navigation
- [ ] Offline indicators i zachowanie przy reconnect
